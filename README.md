## Instalação e execução

1. Abra um terminal na pasta onde estão os arquivos do projeto.  
2. Crie e ative o ambiente virtual:
   python3 -m venv venv
   source venv/bin/activate


3.Instale as dependências:

pip install -r requirements.txt

4.Execute o exemplo:

python3 b_1.py
